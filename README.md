# Week1 - The F2E 活動網站設計
# 特別感謝：設計師[KT](https://2022.thef2e.com/users/12061579703802991521)


啟動方式
```
$ npm install
```
```
$ ng serve -o
``` 


### 版本
- angular/cli: 12.1.4
- npm : 6.14.17

### 主要資料夾說明
- pages : 網站中所有頁面
- components : 元件(魔法棒、下滑提示)
- service : 滑鼠效果、頁面位移效果 

### 使用技術
- Angular
- RxJS
- RWD (Support: 1280x720 768x1024 375x667)
- CSS Animate
- 我的小腦袋
